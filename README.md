# Ai-bilder

> AI builder for websites and apps that is cross-platform, supporting multiple AI providers with API keys, domain management, and more.

## Overview

Ai-bilder is a no-code AI platform that enables users to build and deploy AI-powered applications without writing code. The platform provides:

- **Flow-based AI Builder**: Visual interface for creating AI workflows
- **Multi-Provider Support**: Integration with multiple AI providers (OpenAI, Anthropic, etc.)
- **BYOM (Bring Your Own Model)**: Support for custom model endpoints
- **API Management**: Built-in API key and domain management
- **Cross-Platform**: Works across web and mobile platforms

## Getting Started

### Prerequisites

- Node.js 18+ or Python 3.9+
- PostgreSQL 14+
- Redis 6+

### Installation

```bash
# Clone the repository
git clone https://github.com/wonderingtribe/Ai-bilder.git
cd Ai-bilder

# Install dependencies (coming soon)
# npm install
# or
# pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Run the application
# npm start
# or
# python main.py
```
