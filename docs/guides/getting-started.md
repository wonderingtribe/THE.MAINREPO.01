## Installation

### Prerequisites

Ensure you have the following installed:

- Node.js 18 or higher
- PostgreSQL 14 or higher
- Redis 6 or higher
- Git

### Clone the Repository

```bash
git clone https://github.com/wonderingtribe/Ai-bilder.git
cd Ai-bilder
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

```bash
cp .env.example .env
# Edit .env with your configuration
```

### Run the Application

```bash
npm run dev
```
